Built with Qt 5.10

Tested and working builds on:
* MacOS
* Windows (MinGW 5.3.0 kit)
* Linux

# Building and running

## MacOS

`brew install qt`

`brew link -f qt`

`git clone https://github.com/borishonman/cydownload.git`

`cd cydownload`

`qmake cydownload.pro`

`make`

Type `open ./cydownload.app` to run.
